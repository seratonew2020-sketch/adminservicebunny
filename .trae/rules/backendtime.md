Backend Debugging Rule
โปรเจกต์นี้เป็นระบบ Backend Node.js/Express

เวลาคุณทำงานกับโปรเจกต์นี้:

1. ให้ตรวจสอบไฟล์ใน backend/src ก่อนเป็นหลัก
2. เวลามี error 500 ให้ตรวจ:
   - server.js
   - routes/
   - controllers/
   - services/
   - utils/
3. ให้ตรวจสอบว่า request body, params, และ DB query ตรงกับ schema
4. บันทึกสถานะไฟล์ที่แก้ไขทุกครั้งที่ทำงาน
5. เวลาคุณเสนอการแก้ ให้บอกว่าแต่ละไฟล์จะเปลี่ยนอะไร
